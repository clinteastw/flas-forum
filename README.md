# Flask forum app
Basic forum application made via Python, Flask, Flask-SQLAlchemy.

### Features

- Auth via session;
- Rooms with room participants component;
- Activity feed;

### Installation

Create virtualenv
```python
virtualenv venv
```

Install dependencies
```python
venv/scripts/activate
pip install -r -requirements.txt
```

Run app
```python
py run.py
```
